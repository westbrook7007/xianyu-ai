## 一、演示环境说明

对外展示前，请先确认演示环境，避免现场翻车。

| 环境 | 适合场景 | 爬取能力 | 数据类型 |
|------|----------|----------|----------|
| **线上 Vercel** | 对外路演、远程观众 | 无本机爬虫时返回**演示数据** | 模拟商品 + 模拟趋势 |
| **本机 localhost** | 深度演示、真实爬取 | 需同时启动爬虫 `8080` | 真实闲鱼数据 |
| **本机 + 闲鱼已登录** | 完整闭环演示 | 真实爬取 20–30 条 | 真实数据 + AI 评分 |

**推荐对外演示组合：**

- **轻量路演（5 分钟）**：只用线上 Vercel，展示 UI + 演示数据 + 智能筛选
- **完整路演（15 分钟）**：本机环境 + 爬虫已登录，走一遍真实爬取

**演示前检查清单：**

- [ ] 网站能正常打开
- [ ] 首页搜索框、导航栏、功能卡片显示正常
- [ ] 本机真实爬取时：已运行 `crawler\start_crawler.bat`
- [ ] 本机真实爬取时：已用 `登录闲鱼.bat` 完成扫码登录
- [ ] 邮件演示时：`.env.local` 已配置 SMTP

## 二、15 分钟对外演示脚本（推荐）

按以下顺序讲解，观众最容易理解产品价值。

| 顺序 | 页面 | 演示动作 | 讲解要点 |
|------|------|----------|----------|
| 1 | 首页 | 展示标题、四大功能卡片 | 「避黄牛、AI 匹配、价格监控、定时提醒」 |
| 2 | 首页 | 输入 `iPhone 15 Pro`，点「开始选品」 | 模拟买家提出需求 |
| 3 | 偏好页 | 填写问卷，展示邮件通知卡片 | 强调「个性化选品」 |
| 4 | 偏好页 | 点「启动爬取」 | 展示爬取进度动画 |
| 5 | 结果页 | 查看商品卡片、排序、筛选 | 展示 AI 分、卖家标签、好价标识 |
| 6 | 结果页 | 切换排序为「价格最低」 | 展示多维筛选能力 |
| 7 | 趋势页 | 点击「查看价格走势」 | 展示 7/30 日折线图与入手建议 |
| 8 | 首页 | 滚动到「热门关注 / 最近搜索」 | 展示价格趋势模块 |
| 9 | 个人中心 | 展示定时任务、低价提醒、偏好模板 | 展示「省心监控」能力 |
| 10 | 收尾 | 回到首页 | 总结：远程输入 → 智能选品 → 结果回传 |

**演示推荐关键词：**

- `iPhone 15 Pro`（数码类，演示数据最全）
- `PS5 光驱版`（趋势图表现好）
- `DJI Mini 4 Pro`（备用）

## 四、接口自动化测试用例

项目内置冒烟脚本，适合演示前快速自检。

**执行命令：**

```bash
# 本机
npm run verify

# 线上
node scripts/verify.mjs https://xianyu-ai.vercel.app
```

| 用例编号 | 接口/逻辑 | 预期结果 |
|----------|-----------|----------|
| API-01 | `GET /` 首页 | HTTP 200 |
| API-02 | `GET /api/crawl` | 返回爬取配置、crawlerConfigured 等 |
| API-03 | `POST /api/crawl` 无 keyword | HTTP 400 |
| API-04 | `POST /api/crawl` 带 keyword | 返回 status + products 或 queued |
| API-05 | `POST /api/crawl` 后台+邮箱 | status=queued 或 SMTP 校验错误 |
| API-06 | `GET /api/trends/batch` | 返回多关键词趋势 |
| API-07 | `GET /api/trends/{keyword}` | 返回 30 日趋势点 |
| API-08 | `GET /api/products?keyword=` | 返回商品列表（演示模式 demo=true） |
| API-09 | `GET /api/crawl/results` 缺参 | HTTP 400 |
| API-10 | `GET /api/crawl/results` 未命中 | status=not_found |
| LOGIC-01 | 商品按 ID 去重 | 重复 URL 只保留 1 条 |

**验收标准：** 11 项全部 PASS（`npm run verify` 退出码 0）

## 七、已知限制（对外讲解时可主动说明）

| 限制 | 说明 | 建议话术 |
|------|------|----------|
| 线上为演示模式 | Vercel 未接本机爬虫 | 「线上展示完整产品流程，真实爬取在本机环境」 |
| 真实爬取需电脑在线 | 爬虫在运营者电脑执行 | 「正式使用时，由运营者电脑代为查询闲鱼」 |
| 单次爬取 3–10 分钟 | 20–30 条，间隔 ≥30 秒 | 「轻量合规爬取，保证账号安全」 |
| 每日手动上限 3 次 | 防滥用 | 「保护服务稳定性」 |
